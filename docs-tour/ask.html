Ask mode

Ask mode is for understanding code without letting Cursor edit anything. This should feel safe, focused, and easy to enter.

Back to tour hub
  • What this isA read-only mode for architecture questions, code explanation, and codebase exploration.
  • Where to find itOpen the sidepanel with Cmd+I, then press Shift+Tab until Ask is selected.
  • Try it now in CursorKeep this page open, then inspect `playground/taskflow/src/middleware/auth.ts`, `playground/pixelcraft/src/routes/forms.ts`, or `playground/datalens/src/pipeline/aggregate.ts` and ask what is wrong before changing anything.
  • What to noticeNotice that the same chat surface stays in place; you are changing the mode, not leaving the workflow.
