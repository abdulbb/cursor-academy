Agent mode

Agent mode is where Cursor reads files, edits code, and verifies changes. New learners should see this as a reviewable workflow, not a black box.

Back to tour hub
  • What this isThe mode for multi-step edits, verification, and reviewable code changes.
  • Where to find itOpen the sidepanel and select Agent in the mode picker.
  • Try it now in CursorKeep this page open, then make one small, reviewable improvement in your selected playground app and inspect the diff before accepting it.
  • What to noticeNotice the workflow rhythm: prompt, inspect work, review diff, verify result.
