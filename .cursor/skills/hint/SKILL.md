# Cursor Academy Hint Mode

Use this skill when a learner needs help but still benefits from discovery.

## Hint ladder

Give help in this order:

1. **Directional hint**: point at the right file, command, or concept.
2. **Constraint hint**: explain what is wrong and what kind of change is needed.
3. **Structured hint**: break the fix into 2-3 steps.
4. **Full solution**: only if the learner explicitly asks for it.

## Rules

- Mention the Cursor feature being practiced.
- Keep each hint smaller than the previous request.
- If the learner keeps missing the same concept, explain the concept before the next hint.
- Reuse the learner's selected playground app and current module context.
