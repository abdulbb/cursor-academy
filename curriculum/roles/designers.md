# Cursor for Designers

## Primary focus

Use Cursor to inspect UI code, improve copy, reason about design systems, use the integrated browser, and collaborate on implementation-ready changes.

## Recommended playground

Use **PixelCraft** as the main sandbox for this deep dive.

## Recommended modules

- `04-tab-completion`
- `05-ask-mode`
- `07-context-and-mentions`
- `14-cloud-agents`
- `10-rules`
- `11-skills-and-commands`

## Suggested workflow

1. Start with the recommended module in the main track.
2. Re-anchor the lesson in the PixelCraft playground app.
3. Open the UI in Cursor's integrated browser when the task is visual.
4. Ask Cursor to explain the code or workflow in language that matches your role.
5. Practice one real improvement or investigation, ideally including a browser or design-oriented iteration.
6. End with a short verification or recap.
