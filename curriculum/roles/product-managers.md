# Cursor for Product Managers

## Primary focus

Use Cursor to understand codebases, turn product ideas into plans, inspect implementation trade-offs, review what changed, and use Cursor 3's agent-first surface more effectively.

## Recommended playground

Use **PixelCraft** as the main sandbox for this deep dive.

## Recommended modules

- `03-first-ai-chat`
- `08-plan-mode`
- `14-cloud-agents`
- `12-mcp-integrations`
- `18-custom-workflows`

## Suggested workflow

1. Start with the recommended module in the main track.
2. Re-anchor the lesson in the PixelCraft playground app.
3. Use the Agents Window when you want to compare local work versus cloud handoff.
4. Ask Cursor to explain the code or workflow in language that matches your role.
5. Practice one real improvement or investigation.
6. End with a short verification or recap.
