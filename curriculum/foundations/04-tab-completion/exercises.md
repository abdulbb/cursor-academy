# Exercises for Tab Completion and Fast Edits

## Exercise 1

Goal: Use Tab suggestions to make a tiny change in the app-specific file, then explain whether the suggestion sped you up or needed steering.

Run the app test or typecheck command after the edit if the change affects executable code.

Ask the learner: "What did this Cursor feature make easier, and what did you still need to think through yourself?"
