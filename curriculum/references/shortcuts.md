# Cursor Shortcut Reference

## Core chat controls

- `Cmd+I`: open agent side panel
- `Cmd+L`: open chat
- `Shift+Tab`: rotate modes in chat
- `Cmd+.`: open mode menu
- `Cmd+/`: cycle models
- `@`: attach files, folders, symbols, docs, or past chats
- `/`: invoke slash commands such as project skills

## Editing controls

- `Tab`: accept Cursor Tab suggestion
- `Esc`: reject suggestion
- `Cmd+Right`: accept the next word of a suggestion
- `Cmd+K`: inline edit selected code
- `Opt+Return`: question mode from inline edit

## Practical advice

- Use chat for explanation and planning.
- Use Tab for low-friction edits.
- Use Ask mode when you want understanding without changes.
- Use Plan mode before larger changes.
- Use Agent mode when you want Cursor to execute the plan.
