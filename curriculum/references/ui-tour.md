# Cursor UI Tour Reference

This file is the maintenance index for `docs-tour/index.html`.

It answers three questions:

1. Which Cursor Academy module maps to which tour anchor?
2. Which official Cursor docs page is the source?
3. Which remote media URL is embedded in the tour or module README?

## Attribution

Media is embedded from `cursor.com/docs` and `cursor.com/docs-static/...`.

- We do not mirror or redistribute the media files in this repository.
- Copyright remains with Cursor.
- If a URL stops working, update the mapping here and in `docs-tour/index.html`.

## Refresh instructions

When Cursor updates the docs:

1. Open the relevant docs page.
2. Check whether the existing image or video URL still loads.
3. Update the tour section and the matching module `README.md`.
4. Keep the same tour anchor when possible so existing links do not break.
