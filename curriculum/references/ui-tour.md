# Cursor UI Tour Reference

This file is the maintenance index for the per-feature pages in `docs-tour/`.

It answers four questions:

1. Which Cursor Academy module maps to which feature page?
2. Which official Cursor docs page is the source?
3. Which remote media URL is embedded in the feature page and module README?
4. Which page should `/tour <feature>` open?

## Attribution

Media is embedded from `cursor.com/docs` and `cursor.com/docs-static/...`.

- We do not mirror or redistribute the media files in this repository.
- Copyright remains with Cursor.
- If a URL stops working, update the matching feature page in `docs-tour/` and the module `README.md`.

## Refresh instructions

When Cursor updates the docs:

1. Open the relevant docs page.
2. Check whether the existing image or video URL still loads.
3. Update the matching feature page in `docs-tour/`.
4. Update the matching module `README.md`.
5. Keep the page filename stable when possible so `/tour <feature>` keeps working.
