# Cursor Workshop Index

## Cursor 3 anchor

- Cursor 3: unified multi-workspace agent interface, Agents Window, local/cloud handoff, integrated browser, and a smoother path from commit to PR

## Core workshops

- Cursor 101: orientation, Ask/Plan/Debug/Agent, MCP, rules, skills, subagents, cloud agents, Bugbot
- Cursor 201: advanced workflows, context management, plan iteration, marketplace, browser tooling, cloud workflows
- Cursor Rules, Commands, Hooks, and Skills: Cursor customization surface and extension points
- Cursor Admin Guide: rollout, governance, analytics, adoption, privacy, and admin controls
- Model Selection and Token Efficiency: choosing models, splitting planning vs execution, token-aware workflows
- Refactoring Legacy Codebases: plan-first refactors, verification, TDD patterns, hooks, skills
- Cloud Agents: async remote execution, artifacts, parallel work, hosted workflows
- Cursor Automations: triggers, instructions, tools, and recurring agent workflows
- PRD to PR: turning external requirements into plans and implementation

## How to use this index

When a learner finishes a module, point them to the closest matching workshop for deeper examples and live product context. For Cursor 3 specifically, use the Cursor 3 blog post as the product-level overview of the newer interface and agent-first workflow.
