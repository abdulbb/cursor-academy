# Exercises for Automations and Triggered Workflows

## Main exercise

Goal: Design one automation for maintaining this academy repo, such as refreshing workshop metadata or checking example apps.

### Recommended flow

1. Anchor the exercise in a real part of this repo.
2. Explain the advanced feature in one paragraph.
3. Have the learner design or try the workflow.
4. End with a verification or review step.

## Verification

Spell out the trigger, instructions, and validation step for the automation.

## Reflection prompt

Ask the learner: "What would make this feature high leverage for your real workflow, and what would make it overkill?"
