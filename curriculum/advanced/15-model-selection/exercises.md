# Exercises for Model Selection and Token Efficiency

## Main exercise

Goal: Pick one planning task and one implementation task in this repo and recommend different model characteristics for each.

### Recommended flow

1. Anchor the exercise in a real part of this repo.
2. Explain the advanced feature in one paragraph.
3. Have the learner design or try the workflow.
4. End with a verification or review step.

## Verification

Justify the trade-off in terms of reasoning depth, speed, and context needs.

## Reflection prompt

Ask the learner: "What would make this feature high leverage for your real workflow, and what would make it overkill?"
