# Exercises for Context, @-Mentions, and Codebase Search

## Main exercise

Goal: Use `@` references to compare two related files in the selected playground app and ask Cursor how they connect.

### Recommended flow

1. Read the selected playground area relevant to the task.
2. Use the featured Cursor capability for this module.
3. Capture the outcome in one or two sentences.

## Verification

Check that the answer actually used the files you attached rather than guessing from general knowledge.

## Reflection prompt

Ask the learner: "When would you reach for this feature in real work, and when would a simpler approach be enough?"
