# Exercises for Rules for Persistent Project Guidance

## Main exercise

Goal: Inspect this repository's existing rules and propose one small rule improvement for the learning flow.

### Recommended flow

1. Read the selected playground area relevant to the task.
2. Use the featured Cursor capability for this module.
3. Capture the outcome in one or two sentences.

## Verification

Explain why the rule belongs in a rule file instead of being repeated ad hoc in chat.

## Reflection prompt

Ask the learner: "When would you reach for this feature in real work, and when would a simpler approach be enough?"
