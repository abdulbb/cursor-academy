# Exercises for Debug Mode and Reproducing Problems

## Main exercise

Goal: Pick an intentional bug in the selected playground app and ask Cursor to explain how it would reproduce and what evidence to gather.

### Recommended flow

1. Read the selected playground area relevant to the task.
2. Use the featured Cursor capability for this module.
3. Capture the outcome in one or two sentences.

## Verification

Make sure the suggested debugging steps target the actual bug instead of a generic fix pattern.

## Reflection prompt

Ask the learner: "When would you reach for this feature in real work, and when would a simpler approach be enough?"
